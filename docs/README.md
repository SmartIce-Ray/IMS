## 文档维护规则

遵循全局记忆规则（`~/.claude/CLAUDE.md`）：

1. **核心文档保持简洁**: 只包含核心内容，详细内容移至独立文档
2. **专项规则独立管理**: 特定场景的规则单独成文，避免核心文档臃肿
3. **报告按时间归档**: 分析报告和验证报告放在 `reports/` 目录，文件名包含日期
4. **索引及时更新**: 修改文档结构后，更新 `../CLAUDE.md` 的文档索引
