## 核心功能模块

### 1. 成本核算系统

**目标**: 计算产品理论成本，分析毛利率

| 功能 | 说明 | 状态 |
|------|------|------|
| 产品配方管理 | 87个产品配方，BOM多层分解 | ✅ 已完成 |
| 原材料价格管理 | 241种原材料，支持价格历史 | ✅ 已完成 |
| 半成品分解 | 11个半成品递归分解到原材料 | ✅ 已完成 |
| 理论成本计算 | 基于配方自动计算产品成本 | ✅ 已完成 |
| 成本加密 | 敏感成本数据AES加密存储 | ✅ 已完成 |

**核心公式**:
```
理论成本 = Σ(原材料用量 × 原材料单价)
毛利率 = (菜品收入 - 理论成本) / 菜品收入 × 100%
```

## 关键业务指标

### 线上平台核心指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 曝光访问率 | 访问人数 / 曝光人数 | 门店吸引力 |
| 访问购买率 | 购买人数 / 访问人数 | 转化能力 |
| 客单价 | 成交额 / 成交人数 | 人均消费 |
| 单均价 | 成交额 / 订单数 | 每单金额 |

### 成本分析核心指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 理论成本率 | 理论成本 / 菜品收入 | 标准成本占比 |
| 毛利率 | (菜品收入-理论成本) / 菜品收入 | 盈利能力 |
| 成本偏差 | 实际成本 - 理论成本 | 损耗/浪费 |

## 已产出分析成果

### 1. 宁桂杏1958店 vs 野百灵1958店 (6-11月)

| 维度 | 宁桂杏(烤肉) | 野百灵(火锅) |
|------|--------------|--------------|
| 总成交额 | ¥180.8万 | ¥139.5万 |
| 美团占比 | 82.8% | 80.3% |
| 美团转化率 | 10.86% | 8.89% |
| 点评好评率 | 92.99% | 99.14% |
| 订单结构 | 套餐为主(58%) | 代金券为主(63%) |
| 人气榜排名 | 第2名 | 第11名 |

### 2. 核心发现

- **美团是主力渠道**: 两店美团收入占比均超80%
- **宁桂杏转化更强**: 访问→购买率10.86% vs 8.89%
- **野百灵口碑更佳**: 点评好评率99.14%
- **产品策略不同**: 烤肉适合套餐，火锅适合代金券
